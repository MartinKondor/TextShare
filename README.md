# TextShare

Platform to share text, users can up- or down-vote Texts but that's it.

## Skills Required
* Java-Spring
* SQL
* Maven
* Docker
* Travis CI
* REST API
* HTML/CSS/JavaScript
* Bootstrap 5
* React
* Postman
* TDD - Test Driven Development
* IntelliJ IDEA
* Software design
* Database design
* API design
